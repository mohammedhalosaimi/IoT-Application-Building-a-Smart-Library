Recieption Pi Socket¶

class rp_socket.rp_socket¶
connection(user='test')¶

RP Socket is called when a user is logged in. By that, the socket searches for the server IP address/port to connect with. On connecting to the MP User needs to refer to the MP station and follow the attempts. When MP requests for logout, this message is passed to user class to logout.
