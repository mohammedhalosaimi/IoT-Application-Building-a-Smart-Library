Master Pi Web App Controller¶

class mp_controller.RouteController¶
static booklist()¶

The book list route Parameters: (GET) Returns a book list. (POST) Inserts a book Returns: the book_list template populaed by the books in the system.

static data_vis()¶

The data visualisation route Parameters: Returns: the data vis template links to the google studio

static index()¶

The Index Route Parameters: Returns: The index.html rendered template

static login_post()¶

The Login route Parameters: (GET) Returns the login form. (POST) attempts to login Returns: successful login or the login template with error

static logout()¶

The logout route Parameters: Returns: If logged in, will log the user out

static removeBook()¶

The remove book route Parameters: Book ID (Gotten through the form) Returns: Redirects to the booklist

static updateBook()¶

The Update Book Route. Parameters: AddBookForm Returns: Redirects to the booklist
